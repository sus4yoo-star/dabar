# AMOV 디자인 시스템

앱마다 브랜드색은 다르지만, **레이아웃·접근성 원칙은 공통**입니다.

## 공통 원칙 (전 앱)
- **모바일 우선**, 최대 폭 ~440–560px 중앙 정렬.
- **고대비 · 큰 글자 · 큰 탭 타깃** (사용자에 어르신 다수 — 폰 사용).
- **다국어**: 모든 UI 문자열은 i18n 통과, 하드코딩 금지. 긴 번역(독일어·한국어)에도 안 깨지게.
- 흰 배경 + 앱별 포인트색.

## 앱별 브랜드 토큰
| 앱 | 포인트색 | 비고 |
|---|---|---|
| DABAR | 초록 `#58a700` (보조 파랑 `#1f9bef`) | 성장·말씀 |
| SELAH | **딥네이비** `[hex 확정]` | 묵상·고요 |
| MANNA | **딥틸** `[hex 확정]` | `[컨셉]` |
| 새빛교회 곁에 | `[hex]` | `[컨셉]` |

## 공통 컴포넌트 후보 (중복 줄이기)
- 언어 선택기, 토스트, 로그인(소셜+Apple), 계정/개인정보 페이지, 푸시 토글.
- → 앱 간 복붙 대신, 안정화되면 **공용 패키지/템플릿**으로 추출 고려.
