# 개역개정 본문 데이터 (krv)

성경 읽기(`/read`) 화면은 이 폴더의 권별 JSON 을 불러옵니다.

> ⚠️ **저작권**: 성경전서 개역개정판 ⓒ 대한성서공회. **사용 허락을 받은 본문만** 이 폴더에 넣으세요.
> 라이선스 문의: 대한성서공회(bskorea.or.kr) 저작권 사용 신청.

## 파일 위치 / 이름
- 경로: `public/bible/krv/{CODE}.json`
- `CODE` 는 `lib/bibleData.ts` 의 권 코드(`GEN`, `EXO`, … `REV`). 총 66개.
- 파일이 없는 권은 화면에서 "본문 준비 중"으로 표시됩니다(앱은 정상 동작).

## JSON 형식
```json
{
  "code": "GEN",
  "ko": "창세기",
  "chapters": [
    ["1절 본문 …", "2절 본문 …", "…"],   // 1장
    ["1절 …", "2절 …"]                    // 2장
  ],
  "headings": {            // (선택) 소제목 — "장:절" → 제목
    "25:12": "이스마엘의 후예",
    "25:19": "에서와 야곱이 태어나다"
  }
}
```
- `chapters[i]` = (i+1)장, `chapters[i][j]` = (j+1)절 본문(문자열).
- 절 번호는 배열 인덱스로 부여되므로 **빈 절 없이 순서대로** 넣어야 합니다.
- `headings` 는 없어도 됩니다.

## 확인
데이터를 넣은 뒤 `/read` 에서 해당 권을 선택하면 절 목록과 본문이 바로 표시됩니다.
