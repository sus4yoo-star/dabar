# 성경 본문 데이터 (/public/bible)

성경 읽기(`/read`)와 장 전체 병렬 모드가 이 폴더에서 본문을 불러옵니다.

## 폴더 = 역본/언어
- `bible/{버전}/{권코드}.json`
- **`krv`** = 개역개정(한국어, 1열 기본). ⓒ 대한성서공회 — **사용 허락 본문만**.
- 그 외 폴더명은 **언어 코드**: `en`, `th`, `lo`, `zh`, `es`, `pt`, `vi`, `id`, `ms`, `hi`, `bn`, `ja`, `ru`, `fr`, `sw`, `my`, `ar`, `fa`, `ur` …
  - 각 언어의 **공개/허락 역본**만 넣으세요(언어마다 저작권 별도).
- 권코드는 `lib/bibleData.ts` 의 66개 코드(`GEN`…`REV`).

## 병렬 모드
`/read` 에서 **🔀 장 전체 나란히** 를 켜고 상대 언어를 고르면, 1열 `krv` + 2열 `{언어}` 본문을 **절 단위로 나란히** 표시합니다. 데이터가 있는 칸만 채워지고, 없으면 "본문 준비 중"으로 표시됩니다(앱은 정상 동작).

## JSON 형식 (모든 역본 공통)
```json
{
  "code": "JHN",
  "ko": "요한복음",
  "chapters": [
    ["1절 …", "2절 …"],   // 1장
    ["1절 …"]             // 2장
  ],
  "headings": { "3:16": "하나님의 사랑" }   // (선택) 소제목, krv 기준
}
```
- `chapters[i][j]` = (i+1)장 (j+1)절. **빈 절 없이 순서대로**.
- 병렬 정렬은 절 번호(배열 인덱스) 기준이므로, 두 역본의 절 번호 체계가 같아야 깔끔하게 맞습니다.
